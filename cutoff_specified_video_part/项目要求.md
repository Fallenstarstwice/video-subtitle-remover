# 项目功能,
输入的excel文件第一列有要处理的视频的路径,后面的列有要处理的视频的时间戳且实时间戳数量不固定,例如:
```
[{"timestamp_start": "001:13", "timestamp_end": "001:18"}, {"timestamp_start": "015:28", "timestamp_end": "016:51"}]
```

要求:
读取excel文件的视频,并按照时间戳要求使用ffmpeg剪去对应片段.
默认使用绝对路径,若绝对路径找不到则使用相对路径,若找不到文件则提示找不到此文件(显示在excel中的行数,文件路径),并跳过此行处理下一行.
