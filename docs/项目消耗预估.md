# 要求
一个礼拜处理完5000小时视频，大概需要多少颗3090

选取字幕范围:

```yaml
Y : 0.7435185185185185
H : 0.21851851851851853
X : 0.18125
W : 0.65
~       
```
假设输入为1920x1080,24帧每秒, 每秒处理0.4~0.5秒左右,

5000hx60x60=18000000s


一周时间24x7=168h=604800s,

按照 输入视频为1920x1080,24帧/s,选择上面的字幕范围, 处理速度每秒处理为0.5s视频计算:<br> 
(5000h/168h)*(1s/0.5s)=59.52

而按照excel提供的时间戳剪去视频片段(以下简称为`剪视频`)同样需要消耗时间.但由于`剪视频`消耗的是cpu资源,而ai去字幕消耗的是gpu资源,且两者可以并发执行，而且`剪视频`消耗时间大概为`ai去字幕`的1/5(只用一次的数据统计,不是很准),因此只要把`剪视频`剪第一个视频的时间和ai去字幕处理所有视频的时间相加即可求出总时长, 若excel中列出的视频数量较多时,可忽略`剪视频`消耗的时间.

# 测试5880
5880很快，处理同一段视频(40frames/s浮动,206s)用3090(28frames/s,361s)不到2/3的时间，而且占用率还没拉满，同时跑两个`ai去字幕`项目gpu占用率才会满载,此时处理速度下降一些(此时处理视频用时:222s,223s, 40frames/s左右浮动),也就是说能在降速一些的情况下同时处理两个视频<br>
因此设备只要3090的1/4也就是25台即可,但出于冗余,建议多一点.
